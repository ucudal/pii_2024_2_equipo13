\doxysubsection{Detailed Description}
Clase de pruebas para la funcionalidad de la clase Facade. 

La clase {\ttfamily \doxylink{class_ucu_1_1_poo_1_1_discord_bot_1_1_domain_1_1_tests_1_1_facade_tests}{Facade\+Tests}} contiene un conjunto de pruebas unitarias para verificar el correcto funcionamiento de la clase {\ttfamily Facade}. Se asegura de que las operaciones relacionadas con la lista de espera, los entrenadores y las batallas se realicen correctamente.
