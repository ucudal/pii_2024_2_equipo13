\doxysubsection{Detailed Description}
Clase de pruebas para la funcionalidad de la clase Turno. 

La clase {\ttfamily \doxylink{class_library_1_1_tests_1_1_turno_tests}{Turno\+Tests}} contiene un conjunto de pruebas unitarias para verificar el correcto funcionamiento de la clase {\ttfamily Turno}. Se asegura de que los turnos se cambien correctamente entre los jugadores, se validen los ataques especiales y se manejen condiciones especiales como quemaduras o envenenamientos.
