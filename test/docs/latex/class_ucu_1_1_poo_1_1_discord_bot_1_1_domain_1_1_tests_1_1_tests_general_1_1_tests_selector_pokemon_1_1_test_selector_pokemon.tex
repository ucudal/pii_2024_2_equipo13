La clase {\ttfamily \doxylink{class_ucu_1_1_poo_1_1_discord_bot_1_1_domain_1_1_tests_1_1_tests_general_1_1_tests_selector_pokemon_1_1_test_selector_pokemon}{Test\+Selector\+Pokemon}} contiene un conjunto de pruebas unitarias para verificar el correcto funcionamiento de la clase {\ttfamily Selector\+Pokemon}.
