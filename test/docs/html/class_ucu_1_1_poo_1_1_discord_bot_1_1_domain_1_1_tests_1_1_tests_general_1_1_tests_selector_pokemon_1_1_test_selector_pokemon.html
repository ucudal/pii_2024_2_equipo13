Detailed Description

Clase de pruebas para la funcionalidad de SelectorPokemon.

La clase TestSelectorPokemon contiene un conjunto de pruebas unitarias para verificar el correcto funcionamiento de la clase SelectorPokemon. Se asegura de que la selección de Pokémon, la inicialización de la lista de Pokémon disponibles y la presentación de información relacionada se realicen de manera adecuada.
