Detailed Description

Clase de pruebas para la funcionalidad de la lista de espera (WaitList).

La clase WaitListTests contiene un conjunto de pruebas unitarias para verificar el correcto funcionamiento de la clase WaitList, asegurando que los jugadores sean agregados correctamente a la lista de espera y que se obtengan resultados esperados cuando se consultan los entrenadores en espera.
