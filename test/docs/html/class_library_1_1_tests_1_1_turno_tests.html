Detailed Description

Clase de pruebas para la funcionalidad de la clase Turno.

La clase TurnoTests contiene un conjunto de pruebas unitarias para verificar el correcto funcionamiento de la clase Turno. Se asegura de que los turnos se cambien correctamente entre los jugadores, se validen los ataques especiales y se manejen condiciones especiales como quemaduras o envenenamientos.
