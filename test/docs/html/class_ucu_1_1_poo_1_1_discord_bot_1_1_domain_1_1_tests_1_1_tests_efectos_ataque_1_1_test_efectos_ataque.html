Detailed Description

Clase para probar varios efectos de ataque en los Pokemon.

Esta clase proporciona pruebas unitarias para efectos de ataque, como Dormir, Paralizar, Envenenar y Quemar. Cada prueba valida la aplicación, eliminación y estado de activación de estos efectos en un Pokemon.
