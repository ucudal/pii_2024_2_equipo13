Detailed Description

Clase de pruebas para la funcionalidad de la clase Facade.

La clase FacadeTests contiene un conjunto de pruebas unitarias para verificar el correcto funcionamiento de la clase Facade. Se asegura de que las operaciones relacionadas con la lista de espera, los entrenadores y las batallas se realicen correctamente.
