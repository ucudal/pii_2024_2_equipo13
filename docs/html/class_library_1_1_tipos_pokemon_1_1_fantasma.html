Detailed Description

Representa el tipo de Pokémon "Fantasma" y define sus interacciones con otros tipos.

La clase Fantasma implementa la interfaz Itipo y proporciona las propiedades y métodos para verificar las inmunidades, resistencias y debilidades del tipo Fantasma.
