Detailed Description

Representa el tipo de Pokémon "Planta" y define sus interacciones con otros tipos.

La clase Planta implementa la interfaz Itipo y proporciona las propiedades y métodos para verificar las inmunidades, resistencias y debilidades del tipo Planta.
