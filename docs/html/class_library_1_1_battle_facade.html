Detailed Description

Clase que actúa como una interfaz simplificada para manejar las operaciones de la batalla.

La clase BattleFacade proporciona métodos para gestionar la selección de Pokémon, ataques, uso de ítems, y otros elementos necesarios en el flujo de la batalla.
