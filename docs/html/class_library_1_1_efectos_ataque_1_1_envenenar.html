Detailed Description

Clase que representa el efecto de estado "Envenenamiento" en un Pokémon.

La clase Envenenar implementa la interfaz IEfectoAtaque y aplica el efecto de envenenamiento a un Pokémon, causando daño a lo largo de varios turnos.
