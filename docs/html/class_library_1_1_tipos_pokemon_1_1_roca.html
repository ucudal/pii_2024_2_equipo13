Detailed Description

Representa el tipo de Pokémon "Roca" y define sus interacciones con otros tipos.

La clase Roca implementa la interfaz Itipo y proporciona las propiedades y métodos para verificar las inmunidades, resistencias y debilidades del tipo Roca.
