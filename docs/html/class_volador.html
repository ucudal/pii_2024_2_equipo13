Detailed Description

Representa el tipo de Pokémon "Volador" y define sus interacciones con otros tipos.

La clase Volador implementa la interfaz Itipo y proporciona las propiedades y métodos para verificar las inmunidades, resistencias y debilidades del tipo Volador.
