\doxysubsection{Detailed Description}
Clase que gestiona la selección de Pokémon disponibles para los jugadores. 

La clase \doxylink{class_library_1_1_clases_1_1_selector_pokemon}{Selector\+Pokemon} proporciona una lista de Pokémon disponibles para que los jugadores puedan seleccionar sus equipos de batalla.
