\doxysubsection{Detailed Description}
Clase que gestiona el turno de cada jugador en una batalla. 

La clase \doxylink{class_library_1_1_clases_1_1_turno}{Turno} se encarga de alternar los turnos entre dos jugadores, gestionar el estado de finalización del turno y verificar si la batalla ha terminado.
