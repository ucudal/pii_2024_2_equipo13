\doxysubsection{Detailed Description}
Representa un ataque especial de un Pokémon. 

La clase \doxylink{class_library_1_1_ataque_especial}{Ataque\+Especial} implementa la interfaz \doxylink{interface_library_1_1_i_ataque}{IAtaque} y define las propiedades de un ataque especial, como el nombre, el daño, el tipo, la precisión y el efecto.
