\doxysubsection{Detailed Description}
Interfaz que define las propiedades y métodos relacionados con el tipo de un Pokémon. 

La interfaz \doxylink{interface_library_1_1_itipo}{Itipo} se implementa para representar el tipo de un Pokémon y verificar sus interacciones de efectividad con otros tipos.
