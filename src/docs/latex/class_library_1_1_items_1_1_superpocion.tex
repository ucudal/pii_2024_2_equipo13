\doxysubsection{Detailed Description}
Representa un ítem que recupera 70 puntos de vida (HP) de un Pokémon. 

La clase \doxylink{class_library_1_1_items_1_1_superpocion}{Superpocion} implementa la interfaz \doxylink{interface_library_1_1_i_item}{IItem} y proporciona un ítem que permite recuperar una cantidad fija de salud a un Pokémon durante la batalla.
