\doxysubsection{Detailed Description}
Clase que representa el efecto de estado "{}\+Dormir"{} en un Pokémon. 

La clase \doxylink{class_library_1_1_efectos_ataque_1_1_dormir}{Dormir} implementa la interfaz \doxylink{interface_library_1_1_i_efecto_ataque}{IEfecto\+Ataque} y aplica el efecto de dormir a un Pokémon, impidiéndole atacar durante un número determinado de turnos.
