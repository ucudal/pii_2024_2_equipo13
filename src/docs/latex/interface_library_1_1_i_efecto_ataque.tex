\doxysubsection{Detailed Description}
Interfaz que define los métodos y propiedades de un efecto de ataque especial. 

La interfaz \doxylink{interface_library_1_1_i_efecto_ataque}{IEfecto\+Ataque} se implementa para aplicar efectos de estado en los Pokémon durante la batalla.
