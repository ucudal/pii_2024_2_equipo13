\doxysubsection{Detailed Description}
Representa el tipo de Pokémon "{}\+Fuego"{} y define sus interacciones con otros tipos. 

La clase \doxylink{class_fuego}{Fuego} implementa la interfaz Itipo y proporciona las propiedades y métodos para verificar las inmunidades, resistencias y debilidades del tipo \doxylink{class_fuego}{Fuego}.
