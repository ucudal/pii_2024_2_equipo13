\doxysubsection{Detailed Description}
Clase que representa el efecto de estado "{}\+Parálisis"{} en un Pokémon. 

La clase \doxylink{class_library_1_1_efectos_ataque_1_1_paralizar}{Paralizar} implementa la interfaz \doxylink{interface_library_1_1_i_efecto_ataque}{IEfecto\+Ataque} y aplica el efecto de parálisis a un Pokémon, impidiéndole atacar en ciertos turnos.
