\doxysubsection{Detailed Description}
Clase que representa el efecto de estado "{}\+Quemadura"{} en un Pokémon. 

La clase \doxylink{class_library_1_1_efectos_ataque_1_1_quemar}{Quemar} implementa la interfaz \doxylink{interface_library_1_1_i_efecto_ataque}{IEfecto\+Ataque} y aplica el efecto de quemadura a un Pokémon, causando daño adicional en cada turno.
