\doxysubsection{Detailed Description}
Clase que representa el efecto de estado "{}\+Envenenamiento"{} en un Pokémon. 

La clase \doxylink{class_library_1_1_efectos_ataque_1_1_envenenar}{Envenenar} implementa la interfaz \doxylink{interface_library_1_1_i_efecto_ataque}{IEfecto\+Ataque} y aplica el efecto de envenenamiento a un Pokémon, causando daño a lo largo de varios turnos.
