Detailed Description

Representa el tipo de Pokémon "Eléctrico" y define sus interacciones con otros tipos.

La clase Electrico implementa la interfaz Itipo y proporciona las propiedades y métodos para verificar las inmunidades, resistencias y debilidades del tipo Eléctrico.
