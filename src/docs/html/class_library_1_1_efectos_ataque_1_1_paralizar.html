Detailed Description

Clase que representa el efecto de estado "Parálisis" en un Pokémon.

La clase Paralizar implementa la interfaz IEfectoAtaque y aplica el efecto de parálisis a un Pokémon, impidiéndole atacar en ciertos turnos.
