Detailed Description

Representa el tipo de Pokémon "Fuego" y define sus interacciones con otros tipos.

La clase Fuego implementa la interfaz Itipo y proporciona las propiedades y métodos para verificar las inmunidades, resistencias y debilidades del tipo Fuego.
