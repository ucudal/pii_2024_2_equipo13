Detailed Description

Representa el tipo de Pokémon "Veneno" y define sus interacciones con otros tipos.

La clase Veneno implementa la interfaz Itipo y proporciona las propiedades y métodos para verificar las inmunidades, resistencias y debilidades del tipo Veneno.
