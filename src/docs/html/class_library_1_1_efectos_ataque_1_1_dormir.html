Detailed Description

Clase que representa el efecto de estado "Dormir" en un Pokémon.

La clase Dormir implementa la interfaz IEfectoAtaque y aplica el efecto de dormir a un Pokémon, impidiéndole atacar durante un número determinado de turnos.
